About

What GreenSort AI is, and why it exists

📋 Project Overview

GreenSort AI is a web app that identifies waste items from a photo and tells the user exactly how to dispose of them — the right bin, the environmental payoff, and the mistakes to avoid. It was built for the Idea2Impact Hackathon 2026.

❓ Problem Statement

Most recycling contamination isn't malicious — it's confusion. People don't know whether a lined coffee cup is paper or plastic, whether a black tray is recyclable, or where a dead battery belongs. That uncertainty causes entire batches of good recycling to be rejected at sorting facilities. GreenSort AI removes the guesswork by turning "what is this?" into a one-photo question.

⚙️ AI Workflow

  1. User uploads or captures a photo of an item.
  2. The image is passed to a classification model, which returns the detected object, material, and a confidence score.
  3. The material is matched against a disposal knowledge base (category, correct bin, benefits, common mistakes).
  4. Results render instantly, get logged to the user's dashboard, and can be exported as a shareable report.

This MVP ships with a rule-based mock classifier so the full flow works end-to-end without a trained model. It's built as a drop-in slot — see js/data.js — for a real computer-vision model or a multimodal call to the Anthropic API.

🛠 Technologies Used

HTML5 / CSS3 Vanilla JavaScript Chart.js LocalStorage (persistence) Anthropic API (planned)

🚀 Future Scope

  • Swap the mock classifier for a trained image model or the Anthropic API.
  • Location-aware disposal rules (recycling guidelines vary by city).
  • Account sync so dashboard stats persist across devices.
  • Bulk / multi-item detection in a single photo.
